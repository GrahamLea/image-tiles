Screenshots & Demos

This page contains screenshots and demonstrations of Image Tiles.
All the demo applications on this page are included in the Image Tiles Full Release.

If you don't like the look these demos or screenshots, keep in mind that Image Tiles is very flexible. From the size of the tiles, to their appearance, the speed at which they move and the way in which they move on and off the screen - pretty much everything in Image Tiles can be configured or extended.

In the future we're very keen to see new Transitions, which are the way images are moved on and off screen; all the demos below use the Slide Transition. If you have a good idea for a Transition, please send it to us, or, better yet, implement it yourself and send us the code!

Requirements

The demos on this page are executed using Java WebStart.
Java WebStart is installed when you install a Java Runtime.
If you don't have a Java Runtime installed, download one from java.com now!

Simplest Demo

Simplest Demo shows the basic functionality of Image Tiles by animating a single image in a window.

Click on the image on the right to see a screenshot of Simplest Demo, or click the button below to run the demo.

Simplest Demo also shows how easy it can be to use Image Tiles within your application. The only line of Image-Tiles code in the SimplestDemo application is:

   AnimatedTileCanvas tileCanvas = new AnimatedTileCanvas(new Pipeline(image), true);

Inside Application Demo

Inside Application Demo shows how Image Tiles components can be embedded into a Swing or AWT application.

Click on the image on the right to see an animated screenshot of Inside Application Demo, or click the button below to run the demo.

Simple Demo

Simple Demo demonstrates some of the more impressive features of Image Tiles, such as custom rendering of tiles and the ability to search directory structures for images..

Click on the image on the right to see a screenshot of Simple Demo.
Simple Demo cannot currently be run through WebStart, but is included in the Image Tiles release.
