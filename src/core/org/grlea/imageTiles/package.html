The main interfaces and classes that make up the Image Tiles framework.

What are the most important classes here?

The most important classes in the main package are probably:

  • TileSpace: defines the number and organisation of tiles.
  • TileRenderer: determines how individual tiles are drawn.
  • Transition: performs the animated transitions from one TileSet to another.

See the pipeline package for classes used to compose the Image Tiles components into an animated tile space.

See the demo package for exampes of how Image Tiles components can be used.
